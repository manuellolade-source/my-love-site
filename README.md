# My Love Site 💕

A romantic proposal website for Eva!

## 🎉 Features

- **Proposal Page**: Ask Eva to be your distant lover with beautiful animations
- **Celebration Pages**: Two unique celebration experiences based on her response
- **Confetti Animation**: Colorful falling confetti on celebration pages
- **Responsive Design**: Works perfectly on mobile and desktop
- **Smooth Animations**: Beautiful transitions and effects throughout

## 📄 Pages

- `index.html` - Main proposal page
- `yes.html` - Celebration for "Yes, baby daddy" response
- `duhhh.html` - Celebration for "Duhhh" response
- `style.css` - All styling and animations
- `confetti.js` - Confetti particle system
